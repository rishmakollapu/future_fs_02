# FUTURE_FS_02
This is Task 2 of the Future Interns Internship under the Full Stack Web Development track.
## Mini CRM Application

A simple Full Stack Mini CRM project built using:

- React.js
- Node.js
- Express.js
- MongoDB

## Features

- Add Leads
- View Leads
- Update Lead Status
- Store data in MongoDB Atlas

## How to Run

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm start
```
